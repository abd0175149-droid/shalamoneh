# 🥤 تطبيق شلمونة الذكي | Shalmoneh Smart App

## 📌 نظرة عامة (Overview)
تطبيق هاتف ذكي (iOS & Android) مبني باستخدام بيئة **Flutter**. يهدف المشروع إلى رقمنة تجربة عملاء سلسلة عصائر "شلمونة"، وتحويلها من مجرد عملية شراء تقليدية إلى "تجربة ولاء تفاعلية" (Gamified Loyalty Experience) تحاكي المعايير العالمية لتطبيقات المقاهي الكبرى.

## 🎯 الأهداف الاستراتيجية (Strategic Goals)
1. **الولاء والاحتفاظ:** تحويل العميل العابر إلى زبون دائم عبر نظام نقاط (شلمونات) ذكي.
2. **الكفاءة التشغيلية:** تخفيف الضغط على الكاشير وتقليل الطوابير عبر ميزة الطلب المسبق (Order Ahead).
3. **التخصيص العميق:** إعطاء العميل حرية بناء مشروبه بتفاصيل دقيقة.
4. **التوسع العالمي:** دعم شبكة الفروع المتنامية محلياً (الأردن) ودولياً (السعودية، العراق، فلسطين).

## 🛠 حزمة التقنيات (Tech Stack)
* **Frontend:** Flutter (Dart)
* **State Management:** Riverpod / BLoC
* **Backend:** Node.js / Laravel (RESTful API)
* **Database:** PostgreSQL / Firebase
* **Maps:** Google Maps SDK
